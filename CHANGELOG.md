# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.6.1] - 2026-06-19

### Added
- Added an error message for failed password authentication in AuthController.
- Added support for `select` form elements in the form helper.

## [0.6.0] - 2025-01-21

### Added
- Added support for creating number inputs.
- Added loader indicators.
- Added middleware class support.

### Changed
- Updated `webpack` and `npm` package dependencies.
- Improved CRUD listing page, styling, and controllers.
- Cleaned up forms code and standardized data storage.

### Fixed
- Fixed error with empty lists.
- Fixed broken edit/update views.
- Fixed issue with delete breadcrumb trails.

## [0.5.3] - 2020-04-18

### Fixed
- Fixed assets loading issues.

## [0.5.2] - 2020-04-17

### Changed
- Moved to `fahodzi` UI framework.

## [0.5.1] - 2020-04-17

### Changed
- Merged and consolidated master branch changes.

## [0.5.0] - 2020-04-17

### Added
- Added date picker support.
- Added persistent status rendering in JavaScript.
- Added persistent imports support.
- Added key-value store infrastructure.

### Changed
- Reworked WYF controller factory and view instantiation.
- Improved search pagination and lists.
- Simplified model binding.

### Fixed
- Fixed import errors reporting.
